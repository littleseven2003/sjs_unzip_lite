# 测试用例文档

基于 design.md 第 24 节。

## 1. 正常流程

```
root/
├─ sub1/
│  ├─ Archive.7z.001
│  └─ Archive.7z.002
└─ sub2/
   └─ Archive.7z.003
```

**预期：**
- 三个分卷移动到 root
- sub1、sub2 被删除
- Archive.7z.001 解压成功
- 分卷文件删除
- txt 改 rar
- rar 解压
- rar 删除
- 根目录重命名成功

## 2. 初始存在额外文件

```
root/
├─ Archive.7z.001
├─ Archive.7z.002
└─ readme.md
```

**预期：**
- 弹窗警告
- 用户取消则任务停止
- 用户继续则执行后续流程

## 3. 分卷缺失

```
root/
├─ Archive.7z.001
└─ Archive.7z.003
```

**预期：**
- 检测缺失 002
- 阻止任务
- 提示用户补充分卷

## 4. 多组分卷

```
root/
├─ A.7z.001
├─ A.7z.002
├─ B.7z.001
└─ B.7z.002
```

**预期：**
- 弹窗选择处理 A 或 B
- 不自动同时处理

## 5. 多个 txt 文件

```
root/
├─ a.txt
└─ b.txt
```

**预期：**
- 弹窗让用户选择
- 不自动选择

## 6. 密码错误

**预期：**
- 依次尝试密码表
- 全部失败后停止
- 不删除原压缩包
- 日志不输出明文密码

## 7. 最终文件夹名冲突

```
parent/
├─ root/
└─ final_name/
```

用户指定最终文件夹名为 `final_name`。

**预期：**
- 任务开始前或最终重命名前提示冲突
- 不自动覆盖
